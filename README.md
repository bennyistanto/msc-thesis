FLOOD-STREaM: Scientific Educational Tools for Near Real-Time Extreme Rainfall Monitoring. Transforming flood preparedness into **Scalable**, **Near Real-Time**, and **Adaptable Solutions**.

FLOOD-STREaM is one of output from MSc research project at Bogor Agricultural University ([IPB](https://www.ipb.ac.id/)) in Bogor, Indonesia, work of [Benny Istanto](https://benny.istan.to), [Prof. Rizaldi Boer](https://scholar.google.com/citations?user=jTPXEp8AAAAJ) and [Dr. I Putu Santikayasa](https://scholar.google.com/citations?user=DcQ58z8AAAAJ) started in 2023.

This tools uses [Jupyter Book](https://jupyterbook.org/en/stable/intro.html) and accessible via [https://bennyistanto.github.io/msc-project/](https://bennyistanto.github.io/msc-project)
